# 🚀 José Jiménez | Personal Portfolio

Bienvenido al repositorio de mi portafolio personal. Este proyecto no solo es una vitrina de mi trabajo, sino también un reflejo de mi filosofía como desarrollador: código limpio, diseño pixel-perfect y cero errores. 

Como **Software Engineer**, combino tres disciplinas clave para crear productos digitales completos: **Frontend Development**, **QA Automation** y **Graphic/UI Design**.

🔗 **[[Visita mi portafolio en vivo aquí](https://portfolio-pi-nine-29.vercel.app/)]**

## ✨ Características del Proyecto

* **Diseño Interactivo & Responsivo:** Creado desde cero con un enfoque *mobile-first*, escalando perfectamente a pantallas de laptop y escritorio.
* **Animaciones Suaves:** Implementación de transiciones fluidas de CSS y navegación interna con `react-scroll`.
* **Carrusel Dinámico de Proyectos:** Desarrollado con `Swiper.js`, incluyendo efectos de hover que intercambian imágenes de estado estático a colores vivos.
* **Optimizado para Vite:** Estructura estricta de assets, separando correctamente imágenes estáticas en `/public` e imágenes procesadas en `/src/assets` para evitar bloqueos del bundler.

## 🛠️ Stack Tecnológico

Este portafolio fue construido utilizando herramientas modernas para garantizar el mejor rendimiento y la mejor experiencia de desarrollo:

* **Core:** React.js, Vite
* **Estilos:** Tailwind CSS
* **Componentes UI:** Swiper.js (Carrusel de proyectos), React Scroll (Navegación)
* **Diseño UI/UX:** Figma, Adobe Illustrator
